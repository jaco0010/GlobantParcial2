🛠️CODIGO

🔌este programa permite gestionar el inventario de productos de una tienda con las opciones de 

✅Agregar nuevos productos con detalles como nombre, descripción, categoría, proveedor, precio y fecha de vencimiento

✅Eliminar productos existentes seleccionándolos por su índice en la lista

✅Mostrar todos los productos almacenados en el inventario

La aplicación utiliza clases como Producto y colecciones como ArrayList para almacenar y manipular los datos.

🚀 Uso
Al ejecutar la aplicación, se presentará un menú con las siguientes opciones:

Agregar producto: Permite ingresar los detalles de un nuevo producto.

Eliminar producto: Muestra una lista numerada de productos y permite eliminar uno seleccionando su índice.

Mostrar productos: Muestra todos los productos almacenados en el inventario.

Salir: Finaliza la aplicación.
